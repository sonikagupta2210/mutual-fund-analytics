# Mutual Fund Analytics Platform

## Project Overview

This project was developed as part of the Bluestock Fintech Internship. The objective was to build an end-to-end Mutual Fund Analytics Platform capable of processing fund and investor data, performing advanced risk analytics, and generating business insights through interactive dashboards.

## Objectives

* Build an ETL pipeline for mutual fund data processing
* Store and manage data using SQLite
* Perform Exploratory Data Analysis (EDA)
* Analyze fund performance using return-based metrics
* Implement risk analytics including VaR, CVaR, and Sharpe Ratio
* Perform investor cohort and SIP continuity analysis
* Develop a fund recommendation engine
* Create interactive Power BI dashboards

## Technology Stack

* Python
* Pandas
* NumPy
* SQLite
* Matplotlib
* Power BI

## Key Features

* Historical VaR (95%) & CVaR Analysis
* Rolling 90-Day Sharpe Ratio

## Key Findings

* Equity funds delivered the strongest long-term returns.
* SIP participation remained stable and showed consistent growth.
* Sharpe Ratio identified the top risk-adjusted performing funds.
* VaR and CVaR highlighted significant differences in downside risk across schemes.
